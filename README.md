# Task Manager Python

Console task manager written in Python with OOP and JSON storage.

Учебный консольный проект на Python для управления задачами.

Проект создан в рамках изучения Python, объектно-ориентированного программирования, работы с файлами и базовой логики приложений.

## Возможности

- Добавление новых задач
- Просмотр всех задач
- Удаление задачи по номеру
- Отметка задачи как выполненной
- Редактирование задачи
- Просмотр невыполненных задач
- Просмотр выполненных задач
- Сохранение задач в JSON-файл
- Загрузка задач из JSON-файла при запуске программы
- Обработка некорректного пользовательского ввода

## Что используется в проекте

- Python
- Объектно-ориентированное программирование
- Классы и методы

## Как запустить проект

1. Установить Python 3.
2. Скачать или клонировать репозиторий.
3. Открыть папку проекта в терминале.
4. Запустить файл командой:

python task_manager.py

После запуска появится консольное меню:

1 - Добавить задачу  
2 - Показать все задачи  
3 - Удалить задачу  
4 - Отметить выполненной  
5 - Редактировать задачу  
6 - Показать список невыполненных задач  
7 - Показать список выполненных задач  
8 - Выйти  

## Пример задачи

Задача содержит:

- название
- срок выполнения
- приоритет
- статус выполнения

Пример:

Задача: Изучить PyTest  
Срок: 20.07.2026  
Приоритет: высокий  
Статус: не выполнено
